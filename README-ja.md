# flounder
flounderはdialogflowのEntityを作るためのライブラリです。公式ライブラリではありません。
このライブラリはdialogflowの Rest APIを使用しています。

## Installation

### flounder requires:

```
pip install git+https://github.com/miurahr/pykakasi.git
```

### install flounder:

```
pip install flounder
```
